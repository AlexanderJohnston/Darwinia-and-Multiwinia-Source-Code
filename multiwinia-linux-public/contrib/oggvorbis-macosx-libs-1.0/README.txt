These binaries are meant for use on MacOS X.

The "lib" directory contains shared and static libraries for Ogg Vorbis
development. The "include" directory has the necessary C headers, and
there is HTML documentation in the "doc" directory. The "examples" directory
has some uses of the API.

Please visit http://www.xiph.org/ogg/vorbis/ for the latest Ogg Vorbis
technology developments!
